271 ## NTuples
272 # Immutable objects are stored as HDF5 structs and inlined into
273 # parent structures. HDF5 only allows typemax(Unt16) bytes
274 # for struct description. NTuples are the most common offender for
275 # exploding struct size. (e.g. in the form of large StaticArrays)
276 # The definitions below prevent inlining of large NTuples and
277 # convert to an array instead.
278 const NTUPLE_INLINE_THRESHOLD = 10
279
280 function writeas(NT::Type{NTuple{N,T}}) where {N,T}
281 if N > NTUPLE_INLINE_THRESHOLD
282 return Vector{T}
283 else
284 return NT
285 end
286 end
